# MentalHealth_RiskAssessment
A complete machine learning pipeline analyzing mental health data to uncover patterns and assess psychological risk indicators, designed for insight and research, not clinical diagnosis.

## Requirements

- Python 3.13.5

#### Install Python using MiniConda

1) Download and install MiniConda from [here](https://docs.anaconda.com/free/miniconda/#quick-command-line-install)
2) Create a new environment using the following command:
```bash
$ conda create -n credit-risk-assessment
```
3) Activate the environment:
```bash
$ conda activate credit-risk-assessment
```

### (Optional) Setup you command line interface for better readability

```bash
export PS1="\[\033[01;32m\]\u@\h:\w\n\[\033[00m\]\$ "
```

## Installation

### Install the required packages

```bash
$ pip install -r requirements.txt
```
